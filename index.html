Hangman

This simple hangman game utilizes HTML, CSS, Bootstrap, Javascript, and JQuery. I'm proud of this game for its simple yet elegant design (sure, the font is laughable, but its fun), and for the fact that it goes well beyond most of the hangman examples found online. It is also completely responsive. There is plenty of room to refactor some of the JS, which I'll get around to eventually. No really, I will!

Gif Button Generator

In addition to HTML, CSS, and Bootstrap, this app uses JS and JQuery to dynamically create buttons that, once clicked, perform an AJAX get request to dynamically append ten gifs from the "GIPHY" API to the page! Gifs play and pause upon being clicked. Went with a super minimalist "American Apparel" type design on this one.

Neighbourly

This simple "neighbourly" app is a group project designed to help people connect with others in their community. It essentially acts as a "posting board" for members to alert and invite the public to their events. We utilize the google maps API, location geocoding, and Javascript to help users see the events happening at their current location. Firebase Database houses all of the information about the events. Some simple code using the moment.JS library keeps the posting board and map up to date.
